# BJJ-VQA: Brazilian Jiu-Jitsu Visual Q&A

**BJJ-VQA** is an open-source visual Q&A benchmark focused on Brazilian Jiu-Jitsu.

Instead of just recognizing poses, this benchmark tests if a Vision-Language Model (VLM) actually understands BJJ. It uses multiple-choice questions to evaluate if the model can spot common mistakes, correct executions, and the real techniques behind the art.
