## How to attribute

When using frames from this dataset, please attribute the original source. Example:

> Frame from "Armlock X Triangulo Partindo da Guarda Fechada" by Cobrinha BJJ & Fitness, licensed under CC BY 4.0.

## Adding new sources

When adding questions from a new video, add a row to the table above with:
- **Title**: Video title
- **Author**: Channel or creator name
- **Source**: YouTube URL
- **License**: CC license with link to deed
- **Used in**: Question IDs that use this video (e.g., 00005-00008)
